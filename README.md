## Run Locally

Clone the project

```bash
  git clone https://github.com/uiuxarghya/PCC-CS494
```

Go to the project directory

```bash
  cd PCC-CS494
```

Go to the specific folder

```bash
  cd <Folder Name> ...
```

Compile the program

```bash
  gcc <filename>.c -o <filename>
```

Run the program

```bash
  ./<filename>
```
